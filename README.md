# adsearch-express

Бот корпоративного справочника для express.ms.

Бот ищет пользователей и контакты в Active Directory через LDAPS по русским и английским ФИО, показывает список найденных сотрудников и по кнопке "Контакты" выдает карточку сотрудника.

## MVP

- Поиск по ФИО, ФИ, ИО, имени, фамилии и части фамилии.
- Поиск только по русским и английским ФИО.
- Выдача до 5 результатов с предложением уточнить запрос при большом количестве совпадений.
- При точном совпадении выводится только точное совпадение.
- В списке результатов показываются ФИО, должность и подразделение.
- Карточка сотрудника содержит доступные контактные данные и фото из AD, если оно есть.
- Пустые поля в карточке скрываются.
- Карточки кешируются в SQLite на 24 часа.
- Административная команда `/clear_cache` очищает весь кеш.
- Доступ обычных пользователей ограничивается средствами express.ms.
- Production-логи не содержат текст поисковых запросов и персональные данные.

## Локальный запуск

1. Создать `.env` на основе `.env.example`.
2. Запустить приложение:

```bash
docker compose up --build
```

3. Проверить health endpoint:

```bash
curl http://127.0.0.1:${APP_PORT:-8000}/health
```

До завершения Этапа 0 обработка webhook и BotX-отправка работают как каркас: структура реальных webhook, JWT и endpoint исходящих сообщений должны быть подтверждены на тестовой регистрации express.ms.
